# private-DBserver-access
这是一个私有的数据库服务器访问 Skill，用于访问私有的数据库服务器。

### 使用场景：
自家搭建的简易服务器，但是其上有AI自维护的文档数据库（git管理），和对外提供的访问接口（基于 ssh 和 可能更新的由服务器上AI维护的bash脚本）。

### 使用步骤
1. 最佳使用场景：第一次使用前需要本地机器与服务器ssh客户端key已经匹配
2. 用各种agents或者vibe coding工具使用这个工具即可
3. 在使用该工具时，AI 会询问该服务器的地址，port，user等，之后再用便可以简单指代

### 其他事项
1. 目前自搭的远程服务器限制了访问范围，根目录只是该数据库，且自动维护的提供访问接口的脚本不在此根目录下
2. 提供的接口基本都是基于 ssh scp 类的，目前有查看 tree 结构，获取文件内容文件，修改文件等
3. 目前最佳使用方式是：每次使用之前 AI 会自动调用 提供的 help 接口，会酌情更新接口文档，然后用户可自行对话、进行功能调用
